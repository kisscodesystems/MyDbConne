Please move MyDbConne.jar into a safe folder.
Open a command line and navigate into that folder.
Type "java -version" to see the actual version of your java. (min 1.8 is required.)
See that the "java" command points to the correct java executable!
Example command to start this application:
java -cp C:\drivers\OracleJdbc.jar;C:\opensourcejava\MyDbConne\MyDbConne.jar com.kisscodesystems.MyDbConne.Main
(Please change the folder of the database driver and the place of the MyDbConne.jar into your folders.)
Type "java -jar MyDbConne.jar ?" for the usable commands.
Type "java -jar MyDbConne.jar help" for more detailed information.
Type "java -jar MyDbConne.jar application story" if you want to read.
Type "java -jar MyDbConne.jar application describe" if you would like to get more details.
